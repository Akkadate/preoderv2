## ส่วนที่แก้ไขได้

### 1. หัวข้อ Email (Subject)
```typescript
subject: '🎉 ยินดีต้อนรับสู่ PreOrder24!',
```

### 2. ข้อความต้อนรับหลัก
```html
<p>ขอบคุณที่เลือกใช้ <strong>PreOrder24</strong>! บัญชีของคุณพร้อมใช้งานแล้ว...</p>
```

### 3. คู่มือ 4 ขั้นตอน
หาข้อความ `📖 คู่มือเริ่มต้นใช้งาน` แล้วแก้ไขแต่ละขั้นตอน:
```html
<strong>สร้างร้านค้า</strong>
<p>ตั้งชื่อร้าน เลือกประเภท และปรับแต่งหน้าร้านของคุณ</p>
```

### 4. รายละเอียด Package
หาข้อความ `🎁 แพ็คเกจปัจจุบัน`:
```html
<h3>🎁 แพ็คเกจปัจจุบัน: Free Plan</h3>
<ul>
  <li>10 รอบขายต่อเดือน</li>
  <li>50 ออเดอร์ต่อเดือน</li>
  <li>สินค้าไม่จำกัด</li>
</ul>
<p>
  <strong>ต้องการเพิ่มลิมิต?</strong> อัพเกรดเป็น Pro Plan เพียง ฿299/เดือน...
</p>
```

### 5. ข้อมูลติดต่อ
หาข้อความ `💬 ต้องการความช่วยเหลือ?`:
```html
ติดต่อเราได้ที่ support@preorder24.com หรือ LINE: @preorder24
```

## ตัวแปรที่ใช้ได้

| ตัวแปร | ความหมาย | ตัวอย่างผลลัพธ์ |
|--------|----------|----------------|
| `${displayName}` | ชื่อผู้ใช้ | คุณ สมชาย |
| `${dashboardLink}` | ลิงก์ไป Dashboard | https://preorder24.com/admin |
| `${APP_URL}` | URL หลักของเว็บ | https://preorder24.com |

---

## ตัวอย่างการแก้ไข

### เปลี่ยนช่องทางติดต่อ
**ก่อน:**
```html
ติดต่อเราได้ที่ support@preorder24.com หรือ LINE: @preorder24
```

**หลัง:**
```html
ติดต่อเราได้ที่ LINE: @preorder24official หรือ โทร 02-xxx-xxxx
```

### เปลี่ยนรายละเอียด Package
**ก่อน:**
```html
<li>10 รอบขายต่อเดือน</li>
<li>50 ออเดอร์ต่อเดือน</li>
```

**หลัง:**
```html
<li>5 รอบขายต่อเดือน</li>
<li>30 ออเดอร์ต่อเดือน</li>
```

---

## หมายเหตุ

1. **อย่าลบ `${...}`** - นี่คือตัวแปรที่ระบบจะแทนค่าให้อัตโนมัติ
2. **ทดสอบก่อน Deploy** - สมัครใหม่เพื่อดู email ที่ได้รับ
3. **ใช้ inline CSS เท่านั้น** - Email ไม่รองรับ CSS ภายนอก

## ไฟล์ที่เกี่ยวข้อง

| ไฟล์ | หน้าที่ |
|------|--------|
| `lib/email.ts` | ส่ง email ทั้งหมด (verification + welcome) |
| `app/api/auth/verify-email/route.ts` | เรียกใช้ sendWelcomeEmail หลังยืนยัน |
| `.env` | `RESEND_API_KEY` สำหรับส่ง email |
